### Розбиття(Partitioning)
RDB - мануальне(в коді).
CassandraDB - автоматичне на основі хеша від ключа розбиття(Partition Key).

### Масштабування
RDB - вертикальне.
CassandraDB - горизонтальне.

### З'єднання(JOINs)
Немає з'єднань.

### Зовнішній ключ

### Денормалізація
